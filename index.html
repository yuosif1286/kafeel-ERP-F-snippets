VS Code Extension for Front-End Developers

This guide will help you install the Alkafeel ERP Snippets Extension to boost your front-end productivity.

Install via VS Code Marketplace

  1. Open VS Code.
  2. Go to the Extensions tab (Ctrl+Shift+X / Cmd+Shift+X).
  3. Search for: Alkafeel ERP Snippets
  4. Click Install.

Install via Terminal

If you prefer the terminal, you can install it directly using:

code --install-extension yuosifraedhashem.alkafeel-erp-f-snippets
